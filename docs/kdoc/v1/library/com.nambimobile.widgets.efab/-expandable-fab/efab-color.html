efabColor

var efabColor: Int

The color of the ExpandableFab. Default value is the colorAccent value defined in your app theme (styles.xml). If colorAccent is not defined in your app theme, colorSecondary may be used.

Usage of this property is preferred over the inherited setBackgroundColor and backgroundTintList methods.
